# 스마트 냉장고 IoT 모니터링 시스템

STM32·Arduino·Raspberry Pi 이기종 보드를 블루투스·Wi-Fi로 연동한 
독거노인 안전 모니터링 시스템입니다. 대한상공회의소 AI 시스템반도체 
SW개발자 과정에서 2인 팀 프로젝트로 진행했습니다.

## 담당 역할
- Arduino-Raspberry Pi 간 Wi-Fi 클라이언트 통신 코드 및 재연결 로직 추가
- STM32-Arduino 간 블루투스 통신 코드 수정 및 연동
- 자체 프로토콜 기반 명령어 파싱 적용
- MariaDB 연동 및 DB 초기화 절차 수립
- 우선순위 기반 LCD 출력 로직 설계
